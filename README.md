# Game Jam

This repository contains the files for our game jam, we landed on the theme of expansion and through discussions decided to create a game in which the player character expands through different stages (i.e. ages)


# Initial Discussion

We used microsoft whiteboard to brainstorm ideas to a mind map, deciding on features and styles!

So far we've created mockups for the baby level (which will be completed in our first development sprint)
